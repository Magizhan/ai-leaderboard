How to Sync Your Usage One-time setup

▼

Chrome Extension Recommended

Install once, then click the extension icon on your usage page. It reads your identity automatically.

  1. Download & unzip the extension
  2. Go to chrome://extensions
  3. Enable Developer mode (top right toggle)
  4. Click Load unpacked & select the unzipped folder
  5. Navigate to claude.ai/settings/usage
  6. Click the extension icon — done!
